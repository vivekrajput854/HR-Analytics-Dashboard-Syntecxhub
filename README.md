# HR-Analytics-Dashboard-Syntecxhub

## 📝 Project Overview
This project focuses on HR Analytics, designed to provide actionable insights into workforce management.
The dashboard helps visualize key metrics to support data-driven decision-making regarding talent management, attrition, and operational efficiency.

## 🛠 Tools Used
Data Analysis: Python (Pandas), Power Query
Visualization Tool: Power BI Desktop

## 📊 Key Features & Metrics
1.) Workforce Demographics: Provides visual breakdowns of headcount, department distribution, age, and employee tenure.
2.) Attrition Analysis: Monitors attrition rates across departments and roles to identify patterns in employee departures.
3.) Performance Tracking: Visualizes employee performance ratings and their alignment with organizational objectives.
4.) Compensation: Analyzes salary slabs and monthly income levels across the organization.

## 🚀 Key Insights
This dashboard transforms raw employee data into actionable insights, helping the organization identify potential bottlenecks, assess factors contributing to attrition, and plan for future workforce needs.
